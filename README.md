# Film Checker

React проєкт створений за допомогою Vite та Tailwind CSS.

## Встановлення залежностей

```bash
npm install
```

## Запуск проєкту

```bash
npm run dev
```

## Збірка

```bash
npm run build
```

## Попередній перегляд продакшн збірки

```bash
npm run preview
```
